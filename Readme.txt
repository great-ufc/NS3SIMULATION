>>Pré condição<<

Ter o simulador ns3 no computador previamente configurado

>>Instruções para execução<<

Passo 1: Adicione todos os arquivos para a pasta principal do simulador
Passo 2: Mova os arquivos a seguir para src/wifi/model: yans-wifi-channel.cc, yans-wifi-channel.h, wifi-phy.cc, wifi-phy.h
Passo 3: Execute script.py com as entradas 10, 50, 100 e 250 (use "python3 script.py 10 50 100 250")
Passo 4: Após o término da execução do script anterior, execute scriptresults.py
Passo 5: Para plotar os gráficos execute scriptplot.py e scriptplottime.py (você de ter a biblioteca matplotlib instalada)
